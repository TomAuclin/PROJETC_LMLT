\hypertarget{md__r_e_a_d_m_e_autotoc_md5}{}\doxysubsection{\texorpdfstring{{\bfseries{2. Cloner le repository}}}{{\bfseries{2. Cloner le repository}}}}\label{md__r_e_a_d_m_e_autotoc_md5}
Pour obtenir une copie locale du projet \+:


\begin{DoxyEnumerate}
\item Accédez à l\textquotesingle{}URL du repository Git\+Hub.
\item Copiez le lien HTTPS ou SSH.
\item Dans un terminal, exécutez \+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ <URL\_DU\_REPOSITORY>}

\end{DoxyCode}
 Par exemple \+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/votre\_nom\_utilisateur/Projet-\/C.git}

\end{DoxyCode}

\item Accédez au dossier cloné \+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ Projet-\/C}

\end{DoxyCode}

\end{DoxyEnumerate}

\end{DoxyEnumerate}

En cas de conflit lors d’un {\ttfamily git pull} ou {\ttfamily git merge} \+:
\begin{DoxyEnumerate}
\item Git indiquera les fichiers en conflit. Ouvrez ces fichiers pour résoudre les différences manuellement.
\item Une fois les conflits résolus, ajoutez les fichiers corrigés \+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ add\ fichier\_conflit}

\end{DoxyCode}

\item Terminez le merge avec un commit \+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ commit\ -\/m\ "{}Conflit\ résolu"{}}

\end{DoxyCode}

\end{DoxyEnumerate}
